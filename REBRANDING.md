# Rebranding: From Wave to Gulin

This document outlines the changes made during the rebranding of the project from "Wave Terminal" to "Gulin Terminal".

## Overview

The project has undergone a complete rebranding to better reflect its evolution and new features. "Gulin" represents a more robust, AI-integrated, and user-centric terminal experience.

## Major Changes

### 1. Naming & Branding
- **Name**: Changed from "Wave" / "Wave Terminal" to **Gulin** / **Gulin Terminal**.
- **Domains**: Updated to `gulin.dev` and related subdomains.
- **Repository**: GitHub organization and repository names updated to reflect Gulin.

### 2. Configuration System
- **Paths**: Configuration files moved/migrated from `~/.wave/` to `~/.gulin/`.
- **Schema**: Updated configuration keys (e.g., `waveai` changed to `gulinai`).
- **Binary**: The CLI tool rebranded from `waveterm` to `gulin` / `wsh`.

### 3. UI/UX Enhancements
- **Gulin AI**: Rebranded and improved context-aware AI assistant.
- **Visual Settings**: A new "General Settings" tab for easier configuration (Language, etc.).
- **Theme**: Updated default theme and logo across the application.

### 4. Internationalization (i18n)
- **Multi-language Support**: Added full support for English and Spanish.
- **Reactive UI**: The application now supports real-time language switching without restarts.

## Key Terminology Changes

| Feature | Old Name | New Name |
|---------|----------|----------|
| Terminal App | Wave Terminal | Gulin Terminal |
| AI Assistant | Wave AI | Gulin AI |
| Configuration | Wave Config | Gulin Config |
| CLI Tool | wsh / waveterm | Gulin (wsh) |

## Documentation
Updated all technical documentation, READMEs, and guides to use the new "Gulin" nomenclature.
