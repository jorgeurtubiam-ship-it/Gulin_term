# GuLiN Terminal

> **🚀 Nueva Funcionalidad:** ¡Internacionalización completa! Ahora disponible en Inglés y Español. Consulta la [Guía de Rebranding](./REBRANDING.md).

Gulin es una terminal de código abierto integrada con IA para macOS, Linux y Windows. Funciona con cualquier modelo de IA. Trae tus propias llaves API para OpenAI, Claude o Gemini, o ejecuta modelos locales a través de Ollama y LM Studio. No requiere cuentas.

Gulin también soporta sesiones SSH duraderas que sobreviven a interrupciones de red y reinicios, con reconexión automática. Edita archivos remotos con un editor gráfico integrado y previsualiza archivos en línea sin salir de la terminal.

## Características Principales

- **Gulin AI**: Asistente de terminal consciente del contexto que lee la salida de tu terminal, analiza widgets y realiza operaciones de archivos.
- **Sesiones SSH Duraderas**: Las sesiones de terminal remota sobreviven a interrupciones de conexión, cambios de red y reinicios de Gulin.
- **Interfaz Arrastrar y Soltar**: Organiza bloques de terminal, editores, navegadores web y asistentes de IA de forma flexible.
- **Editor Integrado**: Edita archivos remotos con resaltado de sintaxis y funciones modernas de edición.
- **Previsualización de Archivos**: Sistema rico para archivos remotos (markdown, imágenes, video, PDFs, CSVs, directorios).
- **Soporte Multi-idioma**: Ahora disponible completamente en **Español** e **Inglés**.

## Instalación

Gulin Terminal funciona en macOS, Linux y Windows.

Puedes encontrar instrucciones específicas para cada plataforma [aquí](https://docs.gulin.dev/gettingstarted).

También puedes descargar Gulin Terminal directamente desde: [www.gulin.dev/download](https://www.gulin.dev/download).
