# 📘 Manual de Uso: Administrador de Plugins

Bienvenido al sistema de auto-equipamiento de Gulin. Este componente te permite gestionar las "habilidades dinámicas" de tu agente sin necesidad de programar.

## 1. Acceso al Administrador
Haz clic en el icono del **rayo azul (bolt)** en la barra lateral derecha. Se abrirá una ventana donde podrás ver todos tus plugins instalados.

## 2. Operaciones Principales
- **Crear Plugin**: Haz clic en `+ Nuevo Plugin`. Se cargará una plantilla base que Gulin o tú podéis completar.
- **Editar**: Selecciona un plugin de la lista lateral para ver su código. Puedes hacer cambios menores y pulsar `Guardar Cambios`.
- **Eliminar**: Usa el icono de la papelera en la lista lateral para quitar una habilidad que ya no necesites.

## 3. Interactuando con la IA
La forma más potente de usar este sistema es dejar que la IA gestione el código:
- **Pedir habilidades**: *"Gulin, crea un plugin que consulte mis facturas en la base de datos y me avise si hay alguna vencida."*
- **Actualizar**: *"Gulin, modifica el plugin de clima para que también me diga la humedad."*

## 4. Notas Importantes
- **Recarga Automática**: No necesitas reiniciar la aplicación cuando guardas un plugin. Gulin lo "aprende" al instante.
- **Errores**: Si un plugin tiene un error en el código, verás una notificación en la consola de depuración de Gulin.
